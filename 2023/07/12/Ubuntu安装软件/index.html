Ubuntu安装软件

发表于2023-07-12|更新于2023-09-21|LinuxUbuntu安装
|阅读量:

apt

dpkg(Debian package)

具体的例子

飞书

飞书官网 (feishu.cn) 进入官网

pycharm

pycharm官网，不用命令行，这样方便挑板本

进入点击tool，点击创建桌面快捷方式，即可创建

clash for window

选择对应的版本，下载完成后，右击提取到此处，左击进入，找到cfw文件，在此目录打开终端，

在profiles 输入自己的URL，完成后，打开Ubuntu设置，网络，网络代理调成手动，

在clash for window中打开 Start with linux 即可完成。
