终端小技巧

linux: /

windows: \

history

显示过去使用过的指令

history | grep xxx

查看终端含有xxx的输入的历史

然后！序号可以选择

nvidia-smi

查看显卡驱动情况

htop

查看内存情况

ls

列出目录

pwd

打印工作目录，输出绝对路径

cd

cp

rm

注意：一旦删除很难恢复

mkdir

man

显示手册

touch

更新指定文件的访问和修改时间

exit

关闭terminal

sudo

super user do

超级用户do

shutdown

echo

打印环境变量

kill

ping

用于检测网络连接

grep

搜索与正则表达式匹配的行并打印它们

wget

从互联网下载
